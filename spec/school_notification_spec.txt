# School Site Notification System - Implementation Specification

## Overview
Add user authentication and notification system to existing school website to allow parents to receive updates for their children's classes.

## Technical Environment
- **Hosting:** LAMP stack (MariaDB)
- **CMS:** Craft CMS v5
- **Backend Module:** New custom module `registration` (PHP)
- **Email Service:** Postmark API (already configured)
- **Frontend:** Vanilla JS custom component library with `refresh` helper

## Existing Craft Structure
- **Classes:** Craft Entries
- **Class Updates:** Craft Entries
- **Reminders:** Craft Entries (weekly school-wide updates)
- **Activities:** Craft Entries (monthly/quarterly school-wide detail pages)
- **Users:** Craft native User system (to be extended)

```php
// Get unread count
SELECT COUNT(*) as unread_count
FROM {{%entries}} cu
INNER JOIN {{%user_class_subscriptions}} ucs
  ON cu.id = ucs.classEntryId
LEFT JOIN {{%user_update_read_status}} urs
  ON cu.id = urs.updateEntryId AND urs.userId = ucs.userId
WHERE ucs.userId = :userId
  AND cu.sectionId = :updatesSectionId
  AND urs.id IS NULL
  AND cu.postDate > ucs.dateCreated;

// Get unread updates with details
SELECT
  cu.id,
  cu.title,
  cu.postDate,
  c.title as className
FROM {{%entries}} cu
INNER JOIN {{%entries}} c ON cu.classId = c.id
INNER JOIN {{%user_class_subscriptions}} ucs
  ON c.id = ucs.classEntryId
LEFT JOIN {{%user_update_read_status}} urs
  ON cu.id = urs.updateEntryId AND urs.userId = ucs.userId
WHERE ucs.userId = :userId
  AND cu.sectionId = :updatesSectionId
  AND urs.id IS NULL
  AND cu.postDate > ucs.dateCreated
ORDER BY cu.postDate DESC;
```

```javascript
class LoginForm {
  constructor(element) {
    this.element = element;
    this.form = element.querySelector('form');
    this.emailInput = element.querySelector('input[type="email"]');
    this.submitBtn = element.querySelector('button[type="submit"]');
    this.init();
  }

  init() {
    this.form.addEventListener('submit', (e) => this.handleSubmit(e));
  }

  async handleSubmit(e) {
    e.preventDefault();
    const email = this.emailInput.value;

    // Disable form
    this.submitBtn.disabled = true;
    this.submitBtn.textContent = 'Sending...';

    try {
      const response = await fetch('/registration/auth/send-magic-link', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({ email })
      });

      if (response.ok) {
        window.location.href = '/login/check-email';
      } else {
        alert('Something went wrong. Please try again.');
        this.submitBtn.disabled = false;
        this.submitBtn.textContent = 'Send Magic Link';
      }
    } catch (error) {
      console.error(error);
      alert('Something went wrong. Please try again.');
      this.submitBtn.disabled = false;
      this.submitBtn.textContent = 'Send Magic Link';
    }
  }
}
```

**OnboardingForm Component (`js/components/OnboardingForm.js`):**

```javascript
class OnboardingForm {
  constructor(element) {
    this.element = element;
    this.form = element.querySelector('form');
    this.checkboxes = element.querySelectorAll('input[type="checkbox"]');
    this.submitBtn = element.querySelector('button[type="submit"]');
    this.init();
  }

  init() {
    this.form.addEventListener('submit', (e) => this.handleSubmit(e));

    // Enable/disable submit based on selection
    this.checkboxes.forEach(cb => {
      cb.addEventListener('change', () => this.updateSubmitButton());
    });

    this.updateSubmitButton();
  }

  updateSubmitButton() {
    const anyChecked = Array.from(this.checkboxes).some(cb => cb.checked);
    this.submitBtn.disabled = !anyChecked;
  }

  async handleSubmit(e) {
    e.preventDefault();

    const selectedClasses = Array.from(this.checkboxes)
      .filter(cb => cb.checked)
      .map(cb => parseInt(cb.value));

    this.submitBtn.disabled = true;
    this.submitBtn.textContent = 'Saving...';

    try {
      const response = await fetch('/registration/onboarding/save-subscriptions', {
        method: 'POST',
        headers: {
          'Content-Type': 'application/json',
          'X-CSRF-Token': this.form.querySelector('[name="CRAFT_CSRF_TOKEN"]').value
        },
        body: JSON.stringify({ classIds: selectedClasses })
      });

      if (response.ok) {
        window.location.href = '/dashboard';
      } else {
        alert('Something went wrong. Please try again.');
        this.submitBtn.disabled = false;
        this.submitBtn.textContent = 'Continue';
      }
    } catch (error) {
      console.error(error);
      alert('Something went wrong. Please try again.');
      this.submitBtn.disabled = false;
      this.submitBtn.textContent = 'Continue';
    }
  }
}
```

**NotificationBadge Component (`js/components/NotificationBadge.js`):**

```javascript
class NotificationBadge {
  constructor(element) {
    this.element = element;
    this.badge = element.querySelector('.badge');
    this.init();
  }

  async init() {
    await this.fetchUnreadCount();
    // Refresh every 60 seconds
    setInterval(() => this.fetchUnreadCount(), 60000);
  }

  async fetchUnreadCount() {
    try {
      const response = await fetch('/registration/notifications/unread-count');
      const data = await response.json();
      this.updateBadge(data.count);
    } catch (error) {
      console.error('Failed to fetch unread count:', error);
    }
  }

  updateBadge(count) {
    if (count > 0) {
      this.badge.textContent = count > 99 ? '99+' : count;
      this.badge.style.display = 'inline-block';
    } else {
      this.badge.style.display = 'none';
    }
  }
}
```

**UpdatesList Component (`js/components/UpdatesList.js`):**

```javascript
class UpdatesList {
  constructor(element) {
    this.element = element;
    this.updateItems = element.querySelectorAll('[data-update-id]');
    this.init();
  }

  init() {
    // Set up intersection observer to mark as read when viewed
    const observer = new IntersectionObserver(
      (entries) => this.handleIntersection(entries),
      { threshold: 0.5, rootMargin: '0px' }
    );

    this.updateItems.forEach(item => {
      if (!item.classList.contains('read')) {
        observer.observe(item);
      }
    });
  }

  handleIntersection(entries) {
    entries.forEach(entry => {
      if (entry.isIntersecting) {
        const updateId = entry.target.dataset.updateId;
        this.markAsRead(updateId);
        entry.target.classList.add('read');
      }
    });
  }

  async markAsRead(updateId) {
    try {
      await fetch('/registration/notifications/mark-read', {
        method: 'POST',
        headers: {
          'Content-Type': 'application/json',
          'X-CSRF-Token': document.querySelector('[name="CRAFT_CSRF_TOKEN"]').value
        },
        body: JSON.stringify({ updateId: parseInt(updateId) })
      });

      // Update badge count
      const badge = document.querySelector('[data-component="NotificationBadge"]');
      if (badge && badge._instance) {
        badge._instance.fetchUnreadCount();
      }
    } catch (error) {
      console.error('Failed to mark as read:', error);
    }
  }
}
```

### 1.10 Security Considerations

**Magic Link Security:**
- Use cryptographically secure random tokens (`random_bytes(32)`)
- 15-minute expiration
- Single-use tokens (mark as used after validation)
- Don't reveal if email exists in system
- Clean up expired tokens regularly (scheduled task)

**Authentication:**
- Use Craft's native session management
- Require login for all authenticated routes
- CSRF protection on all POST requests
- Rate limit magic link requests (e.g., 3 per email per hour)

**Data Protection:**
- Users can only see their own subscriptions
- Users can only mark their own updates as read
- Validate all entry IDs exist before saving

### 1.11 User Permissions

Create custom Craft user groups:
- **Parent** - Default for magic link signups, can view subscribed classes
- **Teacher** - Can post updates (use Craft's native permissions)
- **Staff** - Can post reminders/activities
- **Admin** - Full access

Assign "Parent" group automatically on user creation in `actionVerify()`.

### 2.3 Frontend Push Subscription

```javascript
class PushNotificationManager {
  async subscribe() {
    // 1. Register service worker
    const registration = await navigator.serviceWorker.register('/sw.js');

    // 2. Request notification permission
    const permission = await Notification.requestPermission();
    if (permission !== 'granted') return;

    // 3. Subscribe to push
    const subscription = await registration.pushManager.subscribe({
      userVisibleOnly: true,
      applicationServerKey: this.urlBase64ToUint8Array(PUBLIC_VAPID_KEY)
    });

    // 4. Send to server
    await fetch('/registration/push/subscribe', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify(subscription.toJSON())
    });
  }
}
```

### 2.4 Backend Push Service

**PushService.php:**

```php
public function sendPushToUser(int $userId, string $title, string $body, string $url)
{
    // 1. Get all push subscriptions for user
    $subscriptions = PushSubscriptionRecord::find()
        ->where(['userId' => $userId])
        ->all();

    // 2. Send to each device
    foreach ($subscriptions as $sub) {
        try {
            $this->sendPushNotification($sub, $title, $body, $url);
            $sub->lastUsed = new DateTime();
            $sub->save();
        } catch (Exception $e) {
            // If subscription expired, delete it
            $sub->delete();
        }
    }
}

private function sendPushNotification($subscription, $title, $body, $url)
{
    // Use web-push-php library
    $auth = [
        'VAPID' => [
            'subject' => 'mailto:admin@yourschool.com',
            'publicKey' => getenv('VAPID_PUBLIC_KEY'),
            'privateKey' => getenv('VAPID_PRIVATE_KEY'),
        ],
    ];

    $webPush = new WebPush($auth);

    $payload = json_encode([
        'title' => $title,
        'body' => $body,
        'url' => $url
    ]);

    $webPush->sendOneNotification(
        Subscription::create([
            'endpoint' => $subscription->endpoint,
            'keys' => [
                'p256dh' => $subscription->p256dhKey,
                'auth' => $subscription->authKey
            ]
        ]),
        $payload
    );
}
```

### 2.5 Trigger Push on New Update

**Event Handler in Module.php:**

```php
Event::on(
    Entry::class,
    Entry::EVENT_AFTER_SAVE,
    function(ModelEvent $event) {
        $entry = $event->sender;

        // Only for new class updates
        if ($entry->sectionId == $updatesSectionId && !$event->isNew) {
            return;
        }

        // Get all users subscribed to this class
        $users = Module::getInstance()->subscriptionService
            ->getUsersForClass($entry->classId);

        // Send push to each user with push enabled
        foreach ($users as $user) {
            if ($user->pushEnabled) {
                Module::getInstance()->pushService->sendPushToUser(
                    $user->id,
                    "New update in {$entry->className}",
                    $entry->title,
                    "/updates/{$entry->id}"
                );
            }
        }
    }
);
```
